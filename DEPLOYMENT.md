# Military Tees UK - Hostinger Deployment Guide

This website is **100% compatible** with Hostinger's Node.js hosting and is specifically configured for deployment there.

## ✅ Hostinger Compatibility Confirmed

### Architecture
- **Next.js 15** with App Router (fully supported by Hostinger Node.js)
- **Server-Side Rendering (SSR)** - runs as Node.js application
- **API Routes** - all backend functionality via Next.js API routes
- **Static Assets** - images, CSS, JS served from public directory

### Database & Services
- **Supabase** - External managed database (perfect for Hostinger)
- **Stripe** - External payment processing
- **No server dependencies** - everything runs through external APIs

## 📋 Pre-Deployment Checklist

### 1. Environment Variables (Set in Hostinger Panel)
```bash
# Supabase Configuration
NEXT_PUBLIC_SUPABASE_URL=your_supabase_project_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
SUPABASE_SERVICE_ROLE_KEY=your_supabase_service_role_key

# Stripe Configuration
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=pk_live_...
STRIPE_SECRET_KEY=sk_live_...
STRIPE_WEBHOOK_SECRET=whsec_...

# App Configuration
NEXT_PUBLIC_APP_URL=https://militarytees.co.uk
NODE_ENV=production
```

### 2. Domain Configuration
- **Primary Domain**: militarytees.co.uk
- **SSL Certificate**: Auto-generated by Hostinger
- **DNS**: Point to Hostinger nameservers

### 3. Build Configuration
The app uses standard Next.js build process:
```bash
npm run build  # Creates .next/standalone for Node.js hosting
npm start       # Starts production server
```

## 🚀 Deployment Steps

### Step 1: Hostinger Setup
1. **Create Node.js Application** in Hostinger control panel
2. **Select Node.js version**: 18.x or higher
3. **Set application folder**: `public_html/military-tees-uk`

### Step 2: Code Upload
```bash
# Option 1: Git Integration (Recommended)
1. Connect GitHub repository in Hostinger panel
2. Set branch: `main` or `production`
3. Enable auto-deployment on push

# Option 2: Manual Upload
1. Build locally: npm run build
2. Upload entire project folder via File Manager
3. Exclude: node_modules, .git, .env.local
```

### Step 3: Dependencies Installation
Hostinger will automatically run:
```bash
npm install
npm run build
```

### Step 4: Environment Variables
In Hostinger Node.js panel, add all environment variables listed above.

### Step 5: Database Setup
1. **Execute Schema**: Run `database/schema.sql` in Supabase SQL Editor
2. **Execute Functions**: Run `database/functions.sql` in Supabase SQL Editor
3. **Configure RLS**: Row Level Security policies are included
4. **Add Sample Data**: Categories and products will be created

### Step 6: Stripe Configuration
1. **Webhook Endpoint**: `https://militarytees.co.uk/api/webhooks/stripe`
2. **Webhook Events**: 
   - `checkout.session.completed`
   - `payment_intent.succeeded` 
   - `payment_intent.payment_failed`
3. **Test Mode**: Use test keys first, then switch to live
